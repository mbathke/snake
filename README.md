# Snake

Just a simple game with the Bevy Game Engine for pastime. There's nothing ready yet.

## Requirements

* [Bevy](https://bevyengine.org/) Version 0.15.1

## Installation

```sh
$ git clone git@github.com:mbathke/snake.git
```

```sh
$ cd snake
$ cargo build
```

## Running

```sh
$ cargo run
```
